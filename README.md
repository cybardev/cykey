# Cy | key

_pronounce: psyche_

Simple MIDI keyboard on the pentatonic scale.

## Features

- F# major pentatonic notes (black keys on piano): only 5 keys available at a time
- 4 loop layers: can be switched ON/OFF (muted)
- loop layer functions: start/save record, cancel record, remove record
- export all layers as single MIDI file
- instrument: only synth pad
